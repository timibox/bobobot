主播功能
主播数据报表

1. 主播月报 - 可查看主播月报

2. 主播周报 - 可查看主播周报

3. 主播日报 - 可查看主播日报

4. 直播时长 - 可查看直播间当天直播多少小时

5. 直播数据 - 可查看直播间当天明细数据

5. 消费排行 - 可查看直播间当天谁消费最多

6. 弹幕排行 - 可查看直播间当天谁发了最多弹幕

水友相关功能

娱乐功能
粉丝群娱乐功能

1. 主播说 - 可以用主播说话的方式来玩梗

2. DIY表情 - 机器人自动制作主播表情包

3. 改标题 - 发送命令可以自动生成房间标题截图

4. 表情包 - 收集主播所有表情随机发送

群管功能
群管理相关功能

1. 群答题 - 加群机器人自动发送问题给给用户答题

2. 权限分配 - 可以自定义权限设置

3. 超管功能 - 添加超管 可以设置某些功能只允许超管使用

4. 违禁词 - 群内触发违禁词可以设置撤回、禁言或者提出

5. 广告监控 - 监控群内广告 发现立马禁言提出
